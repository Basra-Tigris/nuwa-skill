# 06 人物时间线：教员同志

调研时间：2026-06-03

## 时间线

| 时间 | 事件 | 思维影响 |
|---|---|---|
| 1893 | 出生于湖南湘潭韶山 | 农村社会经验成为其中国实际感的重要来源 |
| 1911-1921 | 接触新思想，参与新文化和早期革命活动 | 从救亡图存转向马克思主义 |
| 1925-1927 | 写《中国社会各阶级的分析》《湖南农民运动考察报告》 | 形成阶级分析、农民问题和调查研究方法 |
| 1927-1935 | 秋收起义、井冈山、中央苏区、长征、遵义会议 | 形成农村根据地、独立自主、从实际出发的革命道路 |
| 1937 | 写《实践论》《矛盾论》 | 系统化认识论与矛盾分析方法 |
| 1938 | 写《论持久战》 | 形成长期战略、阶段转化和人民战争框架 |
| 1942-1945 | 延安整风、中共七大 | 实事求是、群众路线、党的建设成熟化 |
| 1949 | 中华人民共和国成立 | 从革命战略转入国家建设与制度设计 |
| 1956-1957 | 《论十大关系》《关于正确处理人民内部矛盾的问题》 | 探索社会主义建设和社会矛盾分类 |
| 1958-1976 | 社会主义建设探索中出现严重曲折 | 官方历史决议指出晚年错误，特别是否定“文化大革命” |
| 1976 | 逝世 | 进入历史评价与思想体系整理阶段 |
| 1981 | 《关于建国以来党的若干历史问题的决议》通过 | 确立官方评价框架：肯定历史地位与教员同志思想，同时指出晚年错误 |
| 2023 | 《建国以来教员同志文稿》（1-20册）、《教员同志年谱》（1-9卷）出版发行 | 中央党史和文献研究院修订出版，为研究提供更完整基础资料 |

## 最新资料动态

- 2023 年，中央党史和文献研究院增补修订《建国以来教员同志文稿》（1-20册）和《教员同志年谱》（1-9卷），由中央文献出版社出版发行。
- 2024-2026 年，人民网、中国共产党新闻网、求是网、国防部/中国军网继续围绕调查研究、两论、持久战、历史决议等主题发布理论与党史文章。

## 思想阶段

1. 早期调查与农民问题阶段：重视中国社会结构和农村实际。
2. 革命道路形成阶段：独立自主，农村包围城市。
3. 哲学和战略成熟阶段：实践论、矛盾论、持久战、群众路线。
4. 建国后探索阶段：社会主义改造、建设探索、人民内部矛盾。
5. 晚年错误阶段：官方历史决议要求区分教员同志思想科学体系与晚年错误。
